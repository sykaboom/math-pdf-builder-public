- 버그: 요소 메뉴(빈칸/이미지/박스)에서 편집/복사/삭제/빈칸 취소가 동작하지 않던 문제 수정.

- UI: 수식 메뉴에 "빈칸 취소" 버튼 추가(수식 내 개념빈칸 해제).

- 기능: 프롬프트 다운로드 날짜 표시를 수동 입력값으로 고정하고 GitHub API 조회를 제거.
